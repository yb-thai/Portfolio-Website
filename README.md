# Personal Portfolio Website

A modern, fully responsive developer portfolio built with **React** and **Vite**, designed to showcase my Cloud & DevOps engineering experience, AWS/EKS background, and technical projects.  

-  Clean, minimal UI with custom CSS  
-  Mobile-optimized section and responsive layout  
-  Sticky sidebar navigation on desktop  
- Project cards with thumbnails and tech stacks  
- Hosted on GitHub Pages with a custom domain: **boonthai.dev**  

---

## Tech

- **React**
- **Vite**
- **JavaScript (ES6+)**
- **Custom CSS**
- **GitHub Pages (Deployment)**

---

## Features

###  Modern UI/UX  
Styled from scratch with a dark theme, smooth spacing, section headings, and clean typography.

### Fully Responsive Layout  
Mobile-first design with adaptive font sizes, flexible hero layout, and mobile-specific styling.

###  Dynamic Section Navigation  
Desktop includes a sticky sidebar that highlights the active section as you scroll.

### Projects Section  
Each project includes:
- Thumbnail  
- Description  
- Tech stack tags  
- External links
